# 演示材料（作业 §四-3）

本目录用于存放**作业提交用演示**的说明与可选附件。

## 必读

- **录制脚本与分镜**：[demo-recording-guide.md](../evaluation/demo-recording-guide.md)
- **提交达标清单**：[submission-checklist.md](../evaluation/submission-checklist.md)

## 目录说明

| 路径 | 用途 |
|------|------|
| [screenshots/](screenshots/README.md) | 演示截图（按清单命名；可选入 Git，或单独打包提交） |
| [eval-proof.md](eval-proof.md) | 最近一次评测的文字版证明（由 `scripts/export_eval_proof.py` 生成，**可提交**） |

## 视频

若录制 5–10 分钟 MP4，建议命名 `demo-walkthrough.mp4`，可：

- 放在本目录并加入 Git LFS；或
- 上传至网盘 / 作业平台，在根目录 README「演示材料」一节附链接。

勿在视频中展示 API Key 明文。
